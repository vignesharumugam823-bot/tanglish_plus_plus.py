## 📖 Introduction

Tanglish++ is a modern programming language that brings Python's structural elegance to Tamil speakers. By replacing English keywords with Tanglish (Tamil written in Latin script), it minimizes cognitive barriers for native Tamil learners while maintaining production-grade syntax.

All core constructs (conditionals, loops, functions, imports) match Python 3 specifications, making code translation seamless.

## 🚀 Quick Start

### Installation

1. Ensure you have Python 3.6+ installed.
2. Save `tanglish_plus_plus.py` to your project directory.
3. Create a `.t` file with your Tanglish++ code.

### Running a Program

```bash
python tanglish_plus_plus.py your_script.t
